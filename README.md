# Static Web Page Template

## Package Manager
- yarn 1.21.1

## Node Version
- node.js 12.13.1(nodenv)

## Build Tools
- gulp 4.0.2
- @babel/core 7.7.5
- webpack 4.41.2

## Preprocessor
- pug (HTML5)
- stylus (CSS3)
    - Rupture: Media Queries with Stylus

## Setup
```
$ yarn
```

## Gulp

### default
```
$ gulp
```

### build
```
$ gulp build
```

### deploy
```
$ gulp deploy
```
